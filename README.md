# DIP_Semester-Project_100049
 **Pencil_sketch_Project**
 
Simple Python Code to Convert Images into a Pencil Sketch Using OpenCV

  **Overview**

This project uses the **OpenCV** library to apply basic image processing techniques and create **pencil sketch effect** from images. The process includes the following steps:

**1.** Load the original images.

**2.** Convert the images into grayscale.

**3.** Apply **Gaussian blur** to smooth the images.

**4.** Use **Canny edge detection** to find the outlines.

**5.** Invert the detected edges to produce a pencil sketch look.

**6.** Save and display the final sketch images.
